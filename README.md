# ⌨️ Teclado Virtual Circular

Un teclado virtual circular para Windows diseñado para ofrecer una experiencia visual moderna, intuitiva y altamente accesible.

Ideal para pantallas táctiles, configuraciones personalizadas y usuarios que buscan una alternativa innovadora al teclado tradicional.

## ✨ Características

- 🎯 Diseño circular único.
- ⌨️ Escritura en tiempo real sobre cualquier aplicación.
- 🖱️ Compatible con mouse y pantallas táctiles.
- 🚀 Ejecución rápida sin configuraciones complejas.
- 🎨 Interfaz moderna y minimalista.
- 🪶 Aplicación ligera y portátil.
- 💻 Compatible con Windows.

# 🚀 Descarga Rápida

## ⚡ No necesitas instalar Python, dependencias ni compilar código.

Simplemente:

1. Ve a la sección **Releases** del repositorio.
2. Descarga el archivo **.exe** de la última versión.
3. Ejecuta el archivo.
4. ¡Listo! El teclado estará funcionando inmediatamente.
